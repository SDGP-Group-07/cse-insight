# Trading Broker Logos

Place trading broker logo images in this folder.

## Naming Convention

File names should match the broker name in lowercase, with spaces replaced by hyphens.

Examples:
- `acs-capital.png`
- `jb-securities.png`
- `ndb-securities.png`
- `sampath-securities.png`

The `brokerLogoKeyMap` in `tradingBrokers.jsx` maps broker API names to these file keys.
